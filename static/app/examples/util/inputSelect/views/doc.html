InputSelect

输入框信息选择组件, 用于在输入框上处理各种信息选择操作, 其功能如下:

  • 单项信息选择, 即select功能
  • 多项信息选择, 即checkbox功能
  • 树状信息选择, 并提供文本检索功能

调用示例：

  require(["Util/inputSelect"],function(input){
       input({
           id : "inputTest",
           type : "select",
           data : data
       });
  });

Config对象

属性名称 是否必选 说明
id 必选 页面输入框ID
type 必选 单/多选,select/checkbox,默认为select
data 必选 数据源,可以为url或者数据数组
searchAble 可选 是否启用文本搜索功能
callback 必选 当节点选择完毕时的回调函数,传入一个选中id数组
key 可选 自定义键值,替换原有键值
initData 可选 用于初始化的id数组
