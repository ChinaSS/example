功能说明

输入框信息选择组件, 用于在输入框上处理各种信息选择操作, 其功能如下:

  • 单项信息选择, 即select功能
  • 多项信息选择, 即checkbox功能
  • 树状信息选择, 并提供文本检索功能

调用示例：

  require(["Util/inputSelect"],function(input){
       var inputObj = input({
           id : "inputTest",
           type : "checkbox",
           data : [
               {"id":"105001","name":"北京市","checked":true},
               {"id":"105002","name":"河北省"},
               {"id":"105003","name":"河南省","checked":true}
           ],
           searchAble : true,
           initData : [],
           onSelect : function(dataObj){
               console.log(dataObj);
           }
           callback : function(idArray,objArray){
               console.log(idArray);
               console.log(objArray);
           }
       });
  });

API

以InputSelect接收模块返回对象

InputSelect.getInput( id ) 获取指定id的inputSelect实例

假设以inputObj接收该实例

inputObj.refreshPanel( data ) 动态刷新inputPanel面板数据

data为数据对象数组

inputObj.dataInit( initData ) 动态初始化选中数据

initData为数据ID数组

inputObj.getCurrentData() 获取当前组件选中数据

Config对象

属性名称 是否必选 说明
id 必选 页面输入框ID
type 必选 单/多选,select/checkbox,默认为select
data 必选 数据源,可以为url或者数据数组
searchAble 可选 是否启用文本搜索功能
selectNode 可选 单选时,是否可以选择非叶子节点
simpleData 可选 是否使用简单数据结构
panelCss 可选 样式对象,控制数据面板的宽度和对齐位置
key 可选 数据对象字段转换,替换原有字段
initData 可选 用于初始化的id数组
callback 可选 当节点选择完毕时的回调函数,传入参数为当前选中数据的id数组&obj数组
onSelect 可选 当节点选择完毕时的回调函数,传入参数为当前选中数据对象
