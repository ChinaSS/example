功能与实现

  • 用于数据选择的弹出窗口插件
  • 插件是以单体形式实现数据选择窗口,并返回以类别为键值的映射集对象,每个类别对应一个数据对象数组;
  • 内部数据以传入ID为键值存储为一个映射集对象,即数据可存储
  • 使用时需要传入所需类别(权限控制),选择结束后,存储并返回数据映射集
  • 该插件仅对不同类别数据进行筛选,并不对数据的使用范围进行限定,使用者可以对返回的数据进行任意加工

参数列表

参数名称 是否必选 详细描述
id 必选 用于缓存数据的id,通常为当前触发元素id
tagData 必选 用于选择的数据类型
multi 可选 单选or多选
title 可选 弹出框title,说明当前插件功能
height 可选 弹出框body高度,默认为320px
width 可选 弹出框body宽度,默认为800px
hideTag 可选 只有一个标签(类别)时是否隐藏标签
realTimeData 可选 加载左侧树时,是否强制刷新数据
dataRefill 可选 数据初始化时,是否回填上次选中数据
simpleData 可选 替换默认数据映射集对象,返回简单数据数组和数据映射集对象
callback 可选 处理选中数据的回调函数,接受返回数据作为参数

参数使用

DataSelect({
    id : id, 
    multi : false, 
    title : "数据选择",
    height : "320px",
    width : "800px",
    tagData : ["user","role"], //参考可选数据类型 
    hideTag : false,
    realTimeData : false,
    dataRefill : true,
    simpleData : false,
    callback : function(data){} //若simpleData为true,则接受两个参数 
});

可选数据类型

user,role,group,post,dept

API

以DataSelect接受模块返回对象

DataSelect( config ) 调用数据选择插件,初始化配置

config参数详细说明请参照上方参数列表

DataSelect.getData( dataId [, type, id ]) 根据指定条件获取选中数据

DataSelect.getData( dataId ) 获取id为dataId的实例的所有选中数据 DataSelect.getData( dataId, type ) 获取id为dataId的实例的type类型的选中数据 DataSelect.getData( dataId, type, id ) 获取id为dataId的实例的type类型数据中指定id的一条数据
