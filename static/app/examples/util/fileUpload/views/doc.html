BaseDialog

弹出框组件，基于bootstrap的模态框组件。所有弹出框都是基于该组件,比如alert、comfirm、treeDialog等

API

调用样例：

  require(["Util/baseDialog"],function(baseDialog){
     //为弹出框增加操作按钮 
     var buttons = [];
     buttons.push(
         {name:"确定",callback:function(){
             //此处写扩展代码 
 
         }}
     );
     var dialog = baseDialog({
                id:"TestDialog",
                cache:false,                 //是否缓存，默认为true 
                title:"测试窗口",
                width:"400px",
                height:"100px",
                dialogSize:"",               //modal-lg或modal-sm 
                body:"窗口中间内容",
                buttons:buttons
            });
     //可以通过返回的dialog对象调用相关方法 
     dialog.setBody("修改内容");
 })

Config对象

属性名称 是否必选 说明
id 必选 模态框id，同一页面不可重复
cache 必选 是否缓存，默认true
title 必选 dialog标题
width 必选 自定义宽度
height 必选 自定义高度
dialogSize 必选 modal-lg或modal-sm
body 必选 dialog中间部分内容
buttons 必选 操作按钮数组，见buttons

buttons数组

属性名称 是否必选 说明
name 必选 按钮名称
callback 必选 回调函数

dialog相关方法

方法名称 参数 说明
getBody 无 得到dialog的body对象
setBody 字符串或DOM对象 设置body内容
getFoot 无 得到foot
setFoot buttons数组对象 设置foot
setTitle 字符串或DOM对象 设置dialog的标题名称
