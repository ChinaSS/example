Grid

Grid为数据列表组件，遵循AMD标准，依赖boostrap、jquery、grid.css

主要功能点如下：

  1. 自定义Title,工具栏
  2. 异步加载数据,动态刷新
  3. 支持前端排序/后端排序
  4. 支持数据分页显示

样例代码

define(["FilePath"],function(Grid){
    var config = {
        id:"DemoOne",                        //用于缓存的ID,=placeAt 
        placeAt:"DemoGirdDivId",             //存放Grid的容器ID 
        pageSize:10,                         //每页数据条数 
        title:'人员信息列表',                 //表格Title 
        hidden:false,                       //表格是否可隐藏，只显示标题 
        index:"checkbox",                   //首列为单选[radio]还是多选[checkbox],默认checkbox 
        pagination:true,                    //是否启用数据分页 
        cache:false,                        //是否启用缓存 
        realSort:false,                     //是否启用真排序 
        sortParam:{                         //排序参数对象 
            field: null,
            order: "desc"
        },
        formData:{},                        //数据请求的额外参数 
        key:{},                             //前后台字段名转换 
        layout:[                            //表格字段数组 
            {
                name:"Name",
                field:"Field",
                width:"10%",
                align:"center",
                format:function(obj){//obj为当前行数据对象} 
            }
        ],
        toolbar:[                           //自定义工具栏 
            {
                name:"添加",
                class:"fa-plus-circle",
                callback:function(event){}
            }
        ],
        data:[                              //使用已有数据对象 
            {Name:"张三1",Sex:"男",Phone:"123456",Email:"zhangsan@gmail.com",Address:"CQ"}
        ]
        //使用远程url作为数据源时,格式为 data:{type:"URL",value:"url"} 
    };
    Grid(config);
});

API说明

Function调用

Grid(config) 初始化表格,返回Grid对象
Grid.getGrid(id) 获取Grid对象
Grid.dropCache() 清空Grid对象缓存
GridObj.getSelectedRow() 获取当前Grid对象所有选中数据的数组

Config对象

属性 是否可选 说明
id 覆盖 缓存所用id,可再次得到Grid对象,暂由 placeAt 覆盖赋值
placeAt 必选 表格存放的容器id
pageSize 可选 每页数据条数，默认10条
title 可选 表格标题
hidden 可选 是否赋值决定是否在title栏启用收起/展开表格内容功能图标,true收起,false展开
index 可选 每列input选择框类型,radio/checkbox
pagination 可选 是否启用数据分页
cache 可选 是否启用缓存
realSort 可选 是否启用真排序
sortParam 内部 排序参数对象,暂时仅支持内部使用
formData 可选 数据请求的额外参数
key 可选 前后台数据对象字段名转换,与pageinfo中字段名对应,key对象属性格式为,{后端字段:前端字段}
layout 必选 详见layout对象
toolbar 可选 详见toolbar对象
data 必选 详见data对象

layout对象数组

表格列对象数组

属性 是否可选 说明
name 必选 列标题名称
field 可选 列字段名,与后端匹配,设置时该列可排序
width 可选 列宽度,像素/百分比
format 可选 列值格式化函数,接受每一行的数据对象
click 可选 列点击事件处理函数

toolbar对象数组

操作栏按钮对象的数组，

属性 是否可选 说明
name 必选 操作名称
class 必选 Icon图标，从这里选择
callback 必选 点击事件处理函数，通过event.data得到grid对象

data对象数组

data为直接数据源时,其对象的属性与layout数组对象中的field属性一一对应
data为远程数据源地址时，其格式为:data:{type:"URL",value:"url"}
