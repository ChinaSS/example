样例代码

TypeaheadDemo.js

define(["Util/typeahead","jquery"],function(typeahead,$){
    util.typeahead({    
        id : "TypeaheadId",                                 //input ID 
        data : getStaticPath()+"app/data/typeahead.json", //数据地址 
        dataFormat : function(data){                        //数据格式化输出 
            return data+" Formated";
        },
        callback : function(data){                          //回调函数 
            alert(data);
        }
    });
});

API说明

通过javascript函数调用Typeahead:

typeahead(config);

config参数

名称 类型 默认值 描述
id string null 锚点id,在该元素处初始化typeahead
btn string null button名称, 是否赋值判断是否启用button
data object , string null JSON对象或是数据URL地址
lazyMatch boolean true 是否启用延迟匹配,优化查询性能
dataFormat function null 用于自定义每列数据的格式化输出,接受一个数据参数,返回html结构
callback function null 处理选中数据的回调函数,接受一个数据参数
