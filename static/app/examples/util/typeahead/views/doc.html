样例代码

TypeaheadDemo.js

define(["Util/typeahead","jquery"],function(typeahead,$){
    util.typeahead({    
        id : "TypeaheadId",                                 //inputID 
        data : getStaticPath()+"app/data/typeahead.json", //数据地址 
        key:{
            id : "id",
            data : "data"
        },
        dataFormat : [ 
            {key:"id",field:"ID"},
            {key:"data",field:"DATA"}
        ],
        callback : function(data){                          //回调函数 
            alert(data);
        }
    });
});

API说明

通过javascript函数调用Typeahead:

typeahead(config);

config参数

名称 类型 默认值 描述
id string null 锚点id,在该元素处初始化typeahead
btn string "" button名称, 是否赋值判断是否启用button
data object,string null JSON对象或是数据URL地址
lazyMatch boolean true 是否启用延迟匹配,优化查询性能
filter boolean false 是否启用前端匹配筛选
key obj {id:"id",data:"data"} 数据对象字段转换,替换原有字段
dataFormat obj null 自定义每列的数据显示格式的对象数组,每个对象定义一列数据,对象字段key为转换后的字段名
callback function null 处理选中数据的回调函数,接受一个数据参数
