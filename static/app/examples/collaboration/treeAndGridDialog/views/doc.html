API说明

通过javascript函数调用TreeAndGridDialog:

treeAndGridDialog(config);

config参数

名称 类型 默认值 描述
title string null 用于标示特定tree,以缓存不同数据
grid object null 详见Grid参数对象
tree object null 用于转换成ztree参数对象,详见下文tree参数
callback function null 数据回调处理函数,接受一个data参数

tree参数

名称 类型 默认值 描述
isCache boolean true 是否缓存ztree
data object null ztree数据对象
setting object null ztree参数对象,详情见zTree API
